{% load static %} {% comment %} Include inside {% block extra_head %} of any page that renders the address_picker partial. IMPORTANT: address_picker.js loads WITHOUT ``defer``. It registers ``window.addressPicker`` which Alpine's x-data expression on the picker calls at init time. Alpine itself is loaded ``defer`` in _tokens.html. A non-deferred script tag in head blocks the parser and runs before any deferred script, so the factory exists by the time Alpine initialises the picker. Making this ``defer`` creates a race that yields an empty Alpine scope and an invisible picker.

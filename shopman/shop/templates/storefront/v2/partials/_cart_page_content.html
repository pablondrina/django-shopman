{% comment %} v2 cart page inner content — body of the cart page that re-renders on ``cartUpdated from:body``. Hosted by ``storefront/v2/cart.html`` and returned standalone by ``CartContentV2View`` so the page swaps in place without a full reload after every stepper, coupon, or upsell action. Consumes ``CartProjection`` (typed read model). Stepper, delete and upsell use Alpine + native ``fetch`` then dispatch ``cartUpdated`` — ``htmx.ajax`` is avoided here because the cascading reload of this wrapper destroys the source element mid-swap-pipeline, which trips htmx's post-swap cleanup with a TypeError. Coupon forms keep direct ``hx-post`` because their submit element survives the swap. {% endcomment %}
